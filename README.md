## Tentang Repository

Repository ini berisi hasil praktikum dan tugas mata kuliah Data Science dari Pertemuan 1 sampai Pertemuan 7. Seluruh praktikum dikerjakan menggunakan Python melalui Google Colab sebagai media pembelajaran dan penerapan konsep Data Science.

Pada mata kuliah ini saya mempelajari berbagai materi mulai dari pengenalan Data Science, pengolahan data menggunakan Pandas dan NumPy, data cleaning, statistik dasar, visualisasi data, persiapan data untuk machine learning, hingga pembuatan model regresi linear.

---

## Daftar Pertemuan

### Pertemuan 1 - Pengenalan Data Science
Materi dasar mengenai Data Science, peran Data Scientist, dan tahapan CRISP-DM.

### Pertemuan 2 - Python, NumPy, dan Pandas
Mempelajari penggunaan Python dan library Pandas untuk membaca, mengelola, dan menganalisis data.

### Pertemuan 3 - Data Cleaning
Melakukan pembersihan data dengan menangani missing value, data duplikat, dan outlier.

### Pertemuan 4 - Statistik Dasar
Mempelajari mean, median, modus, standar deviasi, skewness, kurtosis, dan korelasi.

### Pertemuan 5 - Visualisasi Data
Membuat berbagai visualisasi data menggunakan Matplotlib dan Seaborn.

### Pertemuan 6 - Data Preparation untuk Machine Learning
Melakukan encoding, train-test split, dan scaling data menggunakan StandardScaler.

### Pertemuan 7 - Regresi Linear
Membangun model Regresi Linear untuk melakukan prediksi serta mengevaluasi performa model menggunakan MAE, RMSE, dan R².
